Comment afficher du code html dans une page html ?

Pour afficher certains caractères il faut utiliser les symboles d'échappement :

  • &lt; donne <
  • &gt; donne >
  • &amp; donne &
  • &quot; donne "

La convention est d'insérer tout code dans une balise <code></code>.

Pour préserver le format du code (espaces, tabulations, sauts de ligne), il faut insérer le code dans une balise <pre></pre>.

Voici du code html.
Affichez la source pour voir la différence. 

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="utf-8" />
    <title></title>
    <link rel="stylesheet" type="text/css" href="css/base.css" />
  </head>
  <body>
    Hello World!
  </body>
</html>
